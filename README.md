<a name="english"></a>
## :gb: English

### Overview

A Python script that easily converts documents (TXT, DOC, DOCX, PDF) into MP3 audio files using Google Text-to-Speech. Perfect for creating audiobooks, accessibility content, or listening to documents on the go.

### Features

- **Multi-Format Support** - TXT, DOC, DOCX, and PDF files
- **Google TTS** - High-quality voice synthesis
- **MP3 Output** - Standard audio format for any device
- **Batch Processing** - Convert multiple documents
- **Easy to Use** - Simple command-line interface

### Supported Formats

| Format | Extension |
|--------|-----------|
| Plain Text | .txt |
| Word Document | .doc, .docx |
| PDF Document | .pdf |

### Quick Start

```bash
git clone https://github.com/fracabu/TextToSpeechProject.git
cd TextToSpeechProject

pip install -r requirements.txt

python text_to_speech.py document.pdf
```

### Requirements

- Python 3.8+
- Google Text-to-Speech API
